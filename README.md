# Feature-Rich JSON Parser & Viewer

A modern, fast, and privacy-friendly JSON parser and viewer built with vanilla HTML, CSS, and JavaScript.

## Features
- **Privacy First**: All processing happens locally in your browser. No data is sent to any server.
- **Tree View**: Interactive, collapsible tree structure for deep JSON navigation.
- **Syntax Highlighting**: Color-coded keys, strings, numbers, booleans, and nulls.
- **Input Tools**: Format, Clear, and Paste from Clipboard.
- **Error Handling**: Detailed error messages for invalid JSON.
- **Responsive Design**: Works on desktop and mobile.
- **Dark Mode**: Sleek, eye-friendly dark interface.

## How to Run
Simply open `index.html` in your web browser. No installation or server required.

## Deployment to GitHub Pages
1. Push this repository to GitHub.
2. Go to **Settings** > **Pages**.
3. Under **Source**, select `main` branch (or `master`) and `/ (root)` folder.
4. Click **Save**.
5. Your site will be live at `https://<your-username>.github.io/<repo-name>/`.
